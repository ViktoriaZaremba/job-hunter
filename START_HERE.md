## ⚡️ Швидкі команди

```bash
# Встановити залежності
npm install

# Запустити dev сервер
npm run dev

# Імпортувати компанії
npm run import:companies

# Імпортувати CSV
npm run import:csv path/to/file.csv your@email.com

# Створити production build
npm run build

# Запустити production
npm start
```

## 🎓 Технології

- **Frontend**: Next.js 14, React 18, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes, NextAuth.js
- **Database**: Supabase (PostgreSQL)
- **Парсинг**: Cheerio
- **Деплой**: Vercel/Netlify готовий

---

## 🔐 Що потрібно налаштувати?

1. **Google OAuth** (15 хв)
   - Google Cloud Console
   - Client ID + Secret

2. **Supabase** (15 хв)
   - Створити проект
   - Виконати SQL
   - Отримати API keys

3. **Environment Variables** (5 хв)
   - Створити .env.local
   - Заповнити credentials
