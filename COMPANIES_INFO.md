## 🚀 Як використовувати

### 1. Імпортувати в базу даних

```bash
cd job-tracker
npm run import:companies
```

Результат:
```
✓ Імпортовано: Grammarly
✓ Імпортовано: MacPaw
✓ Імпортовано: Preply
...
✅ Імпортовано: 294
📊 Всього компаній: 294
```

### 2. Переглянути в додатку

1. Запустіть додаток: `npm run dev`
2. Перейдіть на сторінку "Компанії"
3. Побачите всі 294 компанії
4. Запустіть парсинг вакансій

### 3. Налаштувати автопарсинг

Додаток автоматично збиратиме вакансії з усіх careers сторінок:
- Шукає PM, Product Manager, Project Manager
- Фільтрує за досвідом (3, 4, 5+ років)
- Оновлює щоденно
