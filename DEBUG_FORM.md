## 🧪 Як протестувати:

1. **Перезапустіть сервер**
   ```bash
   # Зупиніть сервер (Ctrl+C)
   npm run dev
   ```

2. **Відкрийте додаток**
   - Перейдіть: http://localhost:3000
   - Увійдіть через Google

3. **Відкрийте консоль браузера**
   - Натисніть F12 або Cmd+Option+I (Mac)
   - Перейдіть на вкладку "Console"

4. **Спробуйте додати заявку**
   - Натисніть "Додати заявку"
   - Заповніть форму:
     - Назва компанії: "MacPaw"
     - Посада: "Product Manager"
   - Натисніть "Додати"

## 🔍 Що перевірити:

### Якщо кнопка не реагує:

1. **Перевірте консоль браузера**
   - Відкрийте F12 → Console
   - Подивіться чи є червоні помилки
   - Скопіюйте текст помилки

2. **Перевірте Network**
   - F12 → Network
   - Натисніть "Додати"
   - Подивіться чи йде запит до `/api/applications`
   - Клікніть на запит → Response
   - Подивіться що повертає сервер

3. **Перевірте базу даних**
   - Відкрийте Supabase Dashboard
   - Table Editor → applications
   - Подивіться чи є записи

### Типові проблеми:

#### ❌ "Unauthorized" (401)
**Проблема**: Не авторизовані  
**Рішення**: 
- Вийдіть та увійдіть знову
- Перевірте чи працює NextAuth
- Перевірте .env.local

#### ❌ "User not found" (404)
**Проблема**: Користувача немає в базі  
**Рішення**:
- API автоматично створює користувача
- Перевірте таблицю `users` в Supabase
- Можливо помилка в схемі БД

#### ❌ "Failed to create application" (500)
**Проблема**: Помилка в базі даних  
**Рішення**:
- Перевірте чи існує таблиця `applications`
- Перевірте чи всі поля правильні
- Подивіться логи в терміналі сервера

#### ❌ Кнопка сіра і не клікається
**Проблема**: Форма в стані завантаження  
**Рішення**:
- Перезавантажте сторінку
- Можливо попередній запит не завершився

#### ❌ Нічого не відбувається
**Проблема**: JavaScript помилка  
**Рішення**:
- Перевірте консоль браузера
- Очистіть кеш (Cmd+Shift+R)
- Перезапустіть сервер

## 🆘 Якщо нічого не допомагає:

1. **Очистіть кеш браузера**
   ```
   Chrome: Cmd+Shift+Delete (Mac) / Ctrl+Shift+Delete (Win)
   Оберіть "Cached images and files"
   ```

2. **Перезапустіть все**
   ```bash
   # Зупиніть сервер
   Ctrl+C
   
   # Очистіть .next
   rm -rf .next
   
   # Перезапустіть
   npm run dev
   ```

3. **Перевірте .env.local**
   ```bash
   cat .env.local
   ```
   Має бути:
   - GOOGLE_CLIENT_ID
   - GOOGLE_CLIENT_SECRET
   - NEXTAUTH_SECRET
   - NEXT_PUBLIC_SUPABASE_URL
   - NEXT_PUBLIC_SUPABASE_ANON_KEY
   - SUPABASE_SERVICE_ROLE_KEY

4. **Перевірте Supabase з'єднання**
   ```bash
   # Відкрийте Node REPL
   node
   
   # Виконайте
   const { createClient } = require('@supabase/supabase-js');
   const supabase = createClient('ваш_url', 'ваш_anon_key');
   supabase.from('users').select('*').then(console.log);
   ```
